Hugo & RelearnTheme 使用

  • 创建Blog 
    hugo new site <new_project>
  • 使用 hugo-theme-relearn 
    git submodule add https://github.com/McShelby/hugo-theme-relearn.git themes/hugo-theme-relearn
  • 创建一级章节 
    hugo new --kind chapter <chapter_title_first>/_index.md
    • 创建二级章节 
      hugo new <chapter_title_first>/<chapter_title_second>/_index.md
  • 服务测试运行
hugo serve
  • 博客生成
hugo
